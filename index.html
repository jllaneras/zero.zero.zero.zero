0.0.0.0 day vulnerabilty test

Run a local web server on port 8080 before loading this page. E.g. python -m http.server 8080. When this page is loaded, 2 HTTP requests are launched, one to http://127.0.0.1:8080?localhost and another one to http://0.0.0.0:8080?zero. Check if any of the requests went through in the web server logs, to see if the web browser is vulnerable to the 0.0.0.0 day vulnerability.
